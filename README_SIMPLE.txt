================================================================================
                    SI CA NE MARCHE PAS
================================================================================

Probleme : "ERR_CONNECTION_REFUSED" ou page inaccessible
---------------------------------------------------------
CAUSE PROBABLE : L'application n'est pas lancee

SOLUTION :
1. Cherchez une fenetre noire avec "Streamlit" dans le titre
2. Si elle n'existe pas : Double-cliquez sur START.bat
3. GARDEZ cette fenetre ouverte
4. Ouvrez votre navigateur sur http://localhost:8501
5. Attendez 15 secondes
6. Actualisez (F5)


Probleme : La fenetre se ferme immediatement
---------------------------------------------
CAUSE : Erreur Python ou Streamlit

SOLUTION :
1. Ouvrez un terminal (cmd) dans ce dossier
2. Tapez : streamlit run dashboard.py
3. Notez l'erreur qui s'affiche
4. Verifiez que Python est installe


Probleme : Page blanche
-----------------------
SOLUTION :
1. Attendez 15-20 secondes (chargement initial)
2. Actualisez la page (F5)
3. Verifiez que la fenetre noire est toujours ouverte
